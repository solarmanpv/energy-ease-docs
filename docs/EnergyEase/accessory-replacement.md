# Accessory Replacement – App User Guide

## 1. Access the Feature

1. Open the app and go to Me. Tap Accessory Replacement to enter the module 
<img src={require("./img/accessory_replacement.png").default} width="240"/>

2. Select the type of accessory supported by the system
<img src={require("./img/select_accessory.png").default} width="240"/>

3. Read the instructions and follow them to complete the accessory replacement process
<img src={require("./img/replace_instructions.png").default} width="240"/>

## 2. Select Device

From the list, select the device that requires accessory replacement (The device must already be added to the current Home)  
<img src={require("./img/select_accessory_device.png").default} width="240"/>


## 3. Perform Replacement

1.  Scan the QR code on the new accessory
<img src={require("./img/scan_accessory_qr_code.png").default} width="240"/>

2. The system will automatically update the accessory profile
<img src={require("./img/updating_accessory_profile.png").default} width="240"/>

3. When “Accessory Replacement Completed” is displayed, the process is successful 
<img src={require("./img/complete_update_profile.png").default} width="240"/>


## 4. Configuration & Data Migration (Optional)

1. The following steps are optional and can be performed later in Migration History
<img src={require("./img/migrate_data.png").default} width="240"/>

2. 【Configuration Migration】
- Use Configuration Migration to sync device settings, including: Parallel settings, Load source, Associated devices
- This step requires a Bluetooth connection
<img src={require("./img/config_migration.png").default} width="240"/>

3. After configuration migration, you can:
- Continue with data migration
- Or skip and perform it later in Migration History
<img src={require("./img/complete_config_migration.png").default} width="240"/>

4. 【Data Migration】
- Data migration is automatically performed in the cloud
- You do not need to stay on the current page
- Progress and status can be viewed in Migration History
<img src={require("./img/data_migration.png").default} width="240"/>

## 5. Special Notes

If the device has not been added to a Home:
- You can follow the on-screen instructions to add it
- Or skip and add it later manually  
<img src={require("./img/add_accessory_device.png").default} width="240"/>

## 6. View Migration Records

1. All configuration migration and data migration records
2. Current migration status  

👉 These can be viewed in Migration History within the app
<img src={require("./img/migration_history.png").default} width="240"/>
