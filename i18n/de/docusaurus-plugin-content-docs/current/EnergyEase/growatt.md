# So erhalten Sie ein Growatt API-Token

Öffnen Sie die ShinePhone-App und gehen Sie zu: **Ich (unten rechts) > Kontoname > API Token**

Tippen Sie auf das Kopiersymbol rechts, um das Token zu erstellen und zu kopieren.

<img src={require("./img/growatt_token.png").default} />

Fügen Sie das Token anschließend in der Energy Ease App ein und wählen Sie das richtige Rechenzentrum aus.

Nachdem Ihr Growatt-Gerät online ist, kann es 5–10 Minuten dauern, bis die Seriennummer (SN) in der Growatt-Geräteliste angezeigt wird.
