# 如何获取 Growatt API Token

在 ShinePhone App 中，依次进入：

**我的（右下角） > 账号名称 > API Token**

点击右侧的复制图标，即可创建并复制 Token。

<img src={require("./img/growatt_token.png").default} />

在 Energy Ease App 中，粘贴该 Token，并选择正确的数据中心。

当 Growatt 设备上线后，可能需要等待 5–10 分钟，才能在 Growatt 设备列表中看到设备序列号（SN）。
