# Comment obtenir un token API Growatt

Dans l’application ShinePhone, accédez à : **Moi (en bas à droite) > Nom du compte > API Token**

Appuyez sur l’icône de copie à droite pour créer et copier le token.

<img src={require("./img/growatt_token.png").default} />

Dans l’application Energy Ease, collez le token et sélectionnez le centre de données approprié.

Une fois votre appareil Growatt en ligne, il peut être nécessaire d’attendre 5 à 10 minutes avant que le numéro de série (SN) apparaisse dans la liste des appareils Growatt.
